**目标：** 对用户 query：① embed query 得向量 → sqlite-vec KNN top-K；② Tantivy 全文 top-K；③ RRF 融合；④ 返回富化后的段落（含 concept_id/path/title/snippet）。

- [ ] **Step 1: retriever.rs**

```rust
//! RAG 检索器：向量 + 全文混合（RRF 融合，ADR-0003）。

use crate::backend::IndexBackend;
use crate::index::tantivy::TantivyIndex;
use crate::llm::EmbedCap;
use crate::search::rrf;
use crate::Result;
use std::sync::Arc;

#[derive(Debug, Clone)]
pub struct RetrievedChunk {
    pub concept_id: String,
    pub path: String,
    pub title: Option<String>,
    pub snippet: String,   // 命中片段（高亮用）
    pub score: f64,
}

pub struct Retriever {
    pub meta: Arc<dyn IndexBackend>,
    pub fulltext: Arc<TantivyIndex>,
    pub sqlite_index: Arc<crate::index::SqliteIndex>,  // 向量查询
    pub embedder: Arc<dyn EmbedCap>,
    pub embed_model: String,
}

impl Retriever {
    pub async fn retrieve(&self, query: &str, k: usize) -> Result<Vec<RetrievedChunk>> {
        // 向量召回
        let qvec = self.embedder.embed(&self.embed_model, &[query.into()]).await?.into_iter().next().unwrap();
        let vec_hits = self.sqlite_index.vector_search(&qvec, k * 2)?;
        // 全文召回
        let ft_hits = self.fulltext.search(query, k * 2)?;

        // RRF 融合（按 concept_id）
        let mut scores: std::collections::HashMap<String, (f64, usize)> = HashMap::new();
        for (rank, h) in vec_hits.iter().enumerate() {
            scores.entry(h.0.clone()).or_insert((0.0, 0)).0 += 1.0 / (60.0 + rank as f64 + 1.0);
        }
        for (rank, h) in ft_hits.iter().enumerate() {
            scores.entry(h.id.clone()).or_insert((0.0, 0)).0 += 1.0 / (60.0 + rank as f64 + 1.0);
            scores.get_mut(&h.id).unwrap().1 = rank;
        }
        let mut merged: Vec<(String, f64)> = scores.into_iter().map(|(id, (s, _))| (id, s)).collect();
        merged.sort_by(|a, b| b.1.partial_cmp(&a.1).unwrap());
        merged.truncate(k);

        // 富化
        let mut out = Vec::with_capacity(merged.len());
        for (id, score) in merged {
            if let Some(row) = self.meta.get_concept(&id)? {
                let snippet = extract_snippet(query, /* body from file or index */ "");
                out.push(RetrievedChunk {
                    concept_id: id,
                    path: row.path,
                    title: row.title,
                    snippet,
                    score,
                });
            }
        }
        Ok(out)
    }
}

fn extract_snippet(_query: &str, body: &str) -> String {
    // 简化：取前 200 字符（M2 改进为含 query 高亮的窗口）
    body.chars().take(200).collect()
}
```

> **gap 标记：** snippet 需要访问 concept body。M1a 的 IndexBackend 不存 body（只存 hash + meta）。**两个方案：** A) retriever 持有 StorageBackend 读文件；B) Tantivy 存 stored field 取 body。**选 B**（Tantivy 已索引，加 stored 字段返回 body 片段更高效）。执行时给 TantivyIndex 的 schema text field 加 `STORED`，search 返回时一并取 body。

## Task 2: 上下文拼装 + token 预算

**Files:**
- Create: `crates/lmnotes-core/src/qa/context.rs`

**目标：** 把 top-K RetrievedChunk 拼成 system prompt 的 context 段，控制总 token 在模型上下文窗口的 50% 以内（留余量给回答）。

- [ ] **Step 1: 实现**

```rust
use super::retriever::RetrievedChunk;

/// 拼装 context 段，每段带编号引用 [1][2]...。
pub fn build_context(chunks: &[RetrievedChunk], max_chars: usize) -> (String, Vec<CitationRef>) {
    let mut sections = Vec::new();
    let mut citations = Vec::new();
    let mut total = 0;
    for (i, c) in chunks.iter().enumerate() {
        let section = format!("[{}] {}\n{}\n", i + 1, c.title.as_deref().unwrap_or(&c.path), c.snippet);
        if total + section.len() > max_chars { break; }
        total += section.len();
        citations.push(CitationRef { index: i + 1, concept_id: c.concept_id.clone(), path: c.path.clone() });
        sections.push(section);
    }
    (sections.join("\n"), citations)
}

#[derive(Debug, Clone)]
pub struct CitationRef {
    pub index: usize,
    pub concept_id: String,
    pub path: String,
}
```

- [ ] **Step 1: chat_stream Tauri 命令（流式 + events）**

```rust
#[tauri::command]
pub async fn chat_stream(
    query: String,
    window: tauri::WebviewWindow,
    retriever: State<'_, Arc<Retriever>>,
    chat: State<'_, Arc<dyn ChatCap>>,
) -> Result<Vec<CitationRef>, String> {
    let chunks = retriever.retrieve(&query, 5).await.map_err(|e| e.to_string())?;
    let (ctx, citations) = build_context(&chunks, 6000);
    let req = ChatRequest { model: ..., messages: vec![
        ChatMessage { role: System, content: format!("{SYSTEM}\n【上下文】\n{ctx}") },
        ChatMessage { role: User, content: query },
    ]};
    let mut stream = chat.chat_stream(req).await.map_err(|e| e.to_string())?;
    while let Some(chunk) = StreamExt::next(&mut stream).await {
        let _ = window.emit("chat-chunk", &chunk.map_err(|e| e.to_string())?);
    }
    Ok(citations)
}
```

import { createSignal, For } from "solid-js";
import { invoke } from "@tauri-apps/api/core";
import { listen } from "@tauri-apps/api/event";

export function ChatDrawer(props: { onClose: () => void; onNavigate: (path: string) => void }) {
  const [input, setInput] = createSignal("");
  const [answer, setAnswer] = createSignal("");
  const [citations, setCitations] = createSignal<CitationRef[]>([]);
  const [streaming, setStreaming] = createSignal(false);

  // 监听 chat-chunk 事件
  listen<string>("chat-chunk", (e) => setAnswer((p) => p + e.payload));

  const ask = async () => {
    setAnswer(""); setStreaming(true);
    try {
      const cites = await invoke<CitationRef[]>("chat_stream", { query: input() });
      setCitations(cites);
    } finally { setStreaming(false); }
  };
  // ... 渲染：输入框、流式回答（[n] 渲染为 Citation 组件）、引用列表
